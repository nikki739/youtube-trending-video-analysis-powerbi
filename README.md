# youtube-trending-video-analysis-powerbi
Power BI project analyzing YouTube trending video data using ETL, DAX, and interactive dashboards.

## Project Overview

This project analyzes YouTube trending video data using Microsoft Power BI. The objective is to identify audience engagement patterns, content performance trends, publishing behavior, and geographic visibility through interactive dashboard reporting.

The project uses ETL preprocessing, DAX calculations, and KPI engineering to transform raw social media data into meaningful business insights.

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Cleaning
- ETL
- Data Visualization
- Dashboard Development

## KPIs Used

- Total Views
- Total Likes
- Total Comments
- Total Engagement
- Engagement Rate
- Average Views per Video

## Dashboard Pages

### 1. Executive Dashboard
Provides a high-level overview of platform performance and engagement metrics.

### 2. Audience & Engagement Dashboard
Analyzes audience interaction using likes, comments, and engagement rate.

### 3. Content Performance Dashboard
Highlights top-performing videos and geographic visibility patterns.

### 4. Conclusion Dashboard
Summarizes findings, recommendations, and future scope.

## Key Insights

- Audience engagement strongly influences trending visibility.
- Videos uploaded during evening hours generally achieved higher engagement.
- Consistent publishing schedules improved content performance.
- Engagement quality provided deeper insights than view counts alone.
- Geographic differences affected audience behavior and visibility.
